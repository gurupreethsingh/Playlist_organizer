About the Platform

A modern Flask and SQLite website for smart song and playlist management

Playlist Organizer is a full-stack music platform where users can upload MP3 songs, play audio directly in the browser, create playlists, and automatically organize music using rich metadata and smart filters.

It is designed to look and feel like a real website with premium spacing, balanced visuals, structured sections, and proper responsive behavior.

What the application offers

The platform provides a complete flow for song uploads, audio playback, playlist management, and organization using language, genre, artist, release year, movie songs, private albums, and singer categories.

It is designed as a cleaner and more realistic full-stack project rather than a plain form-based academic interface.

Built with modular architecture

The application is split into dedicated modules for authentication, user dashboard, songs, playlists, and admin management.

This structure makes the project easier to maintain, scale, and present professionally as a portfolio-grade web application.

User Roles

Two main user types power the application

Role-based access keeps the system organized for both subscribed users and administrators.

Superadmin

Superadmins can manage all users, control subscriptions, access all playlists, review uploaded songs, and handle platform-wide administration.

  • Manage all users
  • Deactivate or reactivate subscriptions
  • Access every playlist
  • Manage all uploaded songs
  • View admin dashboards and statistics

Subscribed User

Users can upload MP3 songs, create playlists, play audio, and organize their personal music collection with smart filters.

  • Upload and store MP3 songs
  • Create and manage playlists
  • Add and remove songs
  • Play music inside the application
  • Use personal dashboard and smart organization
Technology Stack

Built with practical full-stack technologies

The app uses Flask for backend, SQLite for storage, Bootstrap for responsive UI, and browser-based audio playback for MP3 streaming.
